# My Projects

Here is a list of some projects I have worked on. It's a bit sparse right now
as I work on documenting more projects.

- Webgrad:
  A port of Andrej Karpathy's [micrograd](https://github.com/karpathy/micrograd),
  a simple autograd engine, for JavaScript.
  - [GitHub](https://github.com/christopherm99/webgrad)
  - [npm](https://www.npmjs.com/package/webgrad)
- React-Webgrad:
  A frontend for visualizing webgrad's AST in Reactjs.
  - [GitHub](https://github.com/christopherm99/react-webgrad)
  - [Demo](https://christopherm99.github.io/react-webgrad)
